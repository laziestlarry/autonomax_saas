# Ideal Customer Profile

## ICP A: Solo creator / coach
- 1–50k followers, inconsistent income
- Wants an offer and a repeatable system

**Buying triggers:** new audience growth, burnout, missed opportunities

## ICP B: Micro-agency / operator
- 2–10 people, multiple clients, chaotic ops
- Needs a single execution layer (tasks + reporting + billing + proof)

**Buying triggers:** scaling pain, delivery issues, churn
